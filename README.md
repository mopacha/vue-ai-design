# vue-ai-design

> A set of AI components based on Vue

## Installation

#### npm

``` 
npm install vue-ai-design --save

```

## Usage

```
import { AiHighcharts } from 'vue-ai-design'

<ai-highcharts :options="options" ></ai-highcharts>

```
